Data Types

Var

Can be Redeclared

, Let

Can not be Redeclared , Must be declared before use and have block scope.

and Const

Can not be Redeclared, Must be Assigned ,cannot be Reassigned and have Block Scope. 
                        var nameVar = "name";
                        let nameLet = "name";
                        const nameConst = "name";

Can be Redeclared

                        var nameVar = "name";
                        var nameVar = 0;

Can not be Redeclared

                        let nameLet = "name";
                        let nameLet = 0;
                        // SyntaxError: 'nameLet' has been declared

Can not be Reassigned

                            const PI = 3.141592653589793;
                            PI = 3.14;      // This will give an error
                            PI = PI + 10;   // This will also give an error

Must be Assigned

                            const PI = 3.141592653589793;

You can change the elements of a constant array:

                            // You can create a constant array:
                            const cars = ["Saab", "Volvo", "BMW"];

                            // You can change an element:
                            cars[0] = "Toyota";

                            // You can add an element:
                            cars.push("Audi");

But you can NOT reassign the array:

                            const cars = ["Saab", "Volvo", "BMW"];

                            cars = ["Toyota", "Volvo", "Audi"];    // ERROR
